# Utility Scripts

## dns_inventory_gen.py

__Note:__ This will eventually be used to collect ansible inventory from dydns.  

Script is able to convert db.example.zone file into an inventory.yml file.  
```bash
$ ./dns_inventory.py example.com.zone inventory.yml
```
